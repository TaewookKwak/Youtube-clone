# 유튜브 클론 코딩 BY MYSELF
유튜브 클론 코딩하면서 REACT 라이브러리 기본적인 기능에 대해 이해

## Functions
- Main 화면에서 most popular Korean Videos를 불러옴
- Video 시청 가능
- Video owner의 채널 비디오들을 playlist 로 보여줌
- 검색 가능

## Stacks
-React
-PostCSS

## Tools
-Postman
-Netlify
-Git
